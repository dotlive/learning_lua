part_1:命名相关

lua内部模块的API以 luaX_xxx 风格命名：大写字母表识内部模块名；小写字母描述方法名。如：
	luaA_ : lapi.c
	luaB_ :	lbaselib.c
	luaC_ : lgc.c 				(Collection)
	luaD_ : ldo.c 				(Do)
	luaE_ : lstate.c
	luaF_ : lfunc.c 			(Function)
	luaG_ : ldebug.c
	luaH_ : ltable.c 			(Hash)
	luaI_ : lauxlib.c
	luaK_ : lcode.c
	luaL_ : lauxlib.c/h, linit.c (public functions)
	luaM_ : lmem.c 				(Memory)
	luaO_ : lobject.c 			(Object)
	luaP_ : lopcodes.c
	luaS_ : lstring.c 			(String)
	luaT_ : ltm.c 				(Tag)
	luaX_ : llex.c
	luaU_ : ldump.c, lundump.c
	luaV_ : lvm.c 				(Virtual)
	luaX_ : llex.c
	luaY_ : lparser.c 			(Yacc)
	luaZ_ : lzio.c
	lua_  : lapi.c/h + luaconfig.h, debug.c
	luai_ : luaconf.h
	luaopen_ : luaconf.h + libraries (lbaselib.c, ldblib.c, liolib.c, lmathlib.c, loadlib.c, loslib.c, lstrlib.c, ltablib.c)

提供给外部程序使用的API，使用 lua_xxx 的命名风格，定义在lua.h文件中。主要分为以下几部分：
	state操作相关；
	stack操作相关；
	access functions (stack -> c);	// to
	push functions (c -> stack);	// push
	get functions (lua -> stack);	// get/rawget
	set function (stack -> lua);	// set/rawset
	load & call functions <==> load and run lua code.
	coroutine操作相关；

提供给库开发用的API，使用 luaL_xxx 风格命名，定义在lauxlib.h文件中。


part_2:习惯相关

1. lua处理偏移量时，习惯用负数表示从尾部倒数。对栈的索引就是如此，字符串的索引也遵循同一规则。
对栈索引来说，如果为负数的话，top为-1；为正数的话，第一个压入栈的元素索引为1。如：
如果堆栈有 n 个元素，那么索引 1 表示第一个元素（也就是最先被压入堆栈的元素）而索引 n 则指最后一个元素；索引 -1 也是指最后一个元素（即栈顶的元素），索引 -n 是指第一个元素。如果索引在 1 到栈顶之间（也就是，1 ≤ abs(index) ≤ top）我们就说这是个有效的索引。

注：此处可以参考python中的索引处理；换句话，负索引，可以理解为"从后向前数(倒数)"。
